# AI塔罗牌占卜平台 - 项目规格说明书

## 1. 项目概述

### 项目名称
**AI塔罗占卜平台**（TarotQA）

### 项目定位
纯前端静态Web应用，提供塔罗牌占卜服务。用户可直接在浏览器中使用，无需安装任何后端服务。

### 支持平台
- **Web浏览器**: 面向Windows/macOS/Linux用户，支持Chrome、Firefox、Safari、Edge

### 核心价值
- 真实塔罗牌图片 + 专业牌义解读
- 丰富的交互动画效果
- 极低门槛，即开即用
- 单页面应用，无需服务器

## 3. 功能清单

### 3.1 塔罗牌内容（78张）

| 类别 | 数量 | ID范围 | 说明 |
|------|------|--------|------|
| 大阿卡纳 (Major Arcana) | 22张 | 0-21 | 愚者到世界 |
| 小阿卡纳 (Minor Arcana) | 56张 | 22-77 | 权杖(22-35)、圣杯(36-49)、宝剑(50-63)、金币(64-77) |

**完整数据文件**：`resources/tarot-data.json`（78张塔罗牌完整数据）

### 3.2 牌阵功能

| 牌阵名称 | 牌数 | 说明 |
|----------|------|------|
| 单牌阵 | 1 | 快速简单占卜 |
| 三牌阵 | 3 | 过去/现在/未来 |
| 凯尔特十字牌阵 | 10 | 深度详细分析 |
| 爱情金字塔 | 4 | 情感专项 |
| 马蹄铁牌阵 | 7 | 运势综合分析 |

### 3.3 占卜流程

```
用户输入问题 → 选择牌阵 → 洗牌 → 抽牌 → 翻牌 → 牌义展示
```

1. **问题输入**：用户描述想要咨询的问题
2. **牌阵选择**：从上述5种牌阵中选择
3. **洗牌动画**：卡牌洗牌动画效果
4. **抽牌**：点击卡牌抽取
5. **翻牌动画**：卡牌翻转展示正/背面
6. **牌义展示**：显示抽中牌的含义

### 3.4 正位与逆位

每张塔罗牌都有**正位**和**逆位**两种状态：

| 状态 | 出现概率 | 含义特点 |
|------|----------|----------|
| 正位 | 50% | 牌面朝上，含义积极、正面 |
| 逆位 | 50% | 牌面旋转180度，含义复杂、挑战 |

**显示规范**：
- **正位**：图片朝上，文字正常显示
- **逆位**：图片旋转180度，文字保持正常方向（在图片下方，不反转）

**数据规范**：
- 每张牌包含 `description`（正位描述）和 `reversedDescription`（逆位描述）
- 逆位标签显示为红色背景，正位标签显示为绿色背景

### 3.5 每日星座运势

- 覆盖十二星座
- 本地运势数据
- 包含：综合运势、爱情运势、事业运势、财运

### 3.6 AI 深度解读功能

#### 3.6.1 功能描述

在占卜结果页面，用户可点击"AI深度解读"按钮获取 AI 生成的深入分析。

**使用流程**：
1. 用户可使用默认 API Key（部署时通过 GitHub Secrets 配置），无需自行配置
2. 也可在"我的"页面配置自己的 MiniMax API Key（优先级更高）
3. 抽取卡牌后，在结果页面点击"AI深度解读"
4. 系统调用 MiniMax API 获取解读内容
5. 解读结果显示在结果下方

#### 3.6.2 API 调用规范

**端点**: `POST https://api.minimaxi.com/v1/text/chatcompletion_v2`

**请求头**:
```
Authorization: Bearer <API_KEY>
Content-Type: application/json
```

**请求体**:
```json
{
  "model": "MiniMax-M2.7",
  "messages": [
    { "role": "system", "content": "<系统提示词>" },
    { "role": "user", "content": "<用户问题+抽卡信息>" }
  ],
  "stream": false,
  "temperature": 1,
  "top_p": 0.95,
  "max_completion_tokens": 2048
}
```

**响应格式**:
```json
{
  "choices": [{
    "finish_reason": "stop",
    "messages": [
      { "role": "assistant", "content": "<AI回复内容>" }
    ]
  }]
}
```

#### 3.6.2 角色系统

AI 塔罗解读采用角色设定，根据不同场景自动选择合适的顾问角色。

**角色列表**：

| 角色 | 适用场景 | 关键词 |
|------|----------|--------|
| 综合顾问 | 单牌阵、三牌阵、通用问题 | 通用 |
| 职涯发展顾问 | 事业抉择、工作问题 | 工作、事业、跳槽、面试、职场 |
| 爱情情感顾问 | 感情占卜、恋爱问题 | 爱情、感情、恋人、复合、桃花 |
| 财富财务顾问 | 财运占卜、投资理财 | 金钱、财富、投资、理财 |
| 决策指引顾问 | 凯尔特十字、重大决策 | 选择、决定、纠结、两难 |
| 综合运势顾问 | 马蹄铁牌阵、综合运势 | 运势、运气、未来、趋势 |

**角色选择规则**：
1. 优先根据用户问题中的关键词匹配
2. 其次根据牌阵类型自动推荐
3. 无匹配时使用默认"综合顾问"

**角色数据文件**：`resources/tarot-personas.md`

#### 3.6.3 错误处理与调试

**错误场景**：

| 错误场景 | 错误信息 | 处理建议 |
|----------|----------|----------|
| API Key 未配置 | 请先在设置中配置 MiniMax API Key | 引导用户去设置页面 |
| API Key 格式无效 | API Key 格式无效，请检查设置 | 提示格式要求 |
| 网络连接失败 | 网络连接失败，请检查网络后重试 | 检查网络 |
| HTTP 401 | API Key 无效或已过期，请检查设置 | 重新配置 Key |
| HTTP 403 | API Key 权限不足 | 检查 Key 权限 |
| HTTP 429 | 请求过于频繁，请稍后重试 | 限流等待 |
| HTTP 500+ | MiniMax 服务器繁忙，请稍后重试 | 服务端问题 |
| 响应格式错误 | 服务器响应格式错误，请稍后重试 | 重试 |
| AI 返回空内容 | AI 暂时无法提供解读，请稍后重试 | 重试 |

**调试面板**：
- 显示"调试信息"复选框，勾选后展开调试面板
- 调试信息包含：
  - 请求时间戳
  - 牌阵类型
  - 卡牌数量
  - 用户问题
  - 完整请求数据（JSON 格式）
  - 抽卡详情（JSON 格式）

#### 3.6.4 API Key 配置

**默认 API Key**: 通过 GitHub Secrets 配置 (`DEFAULT_MINIMAX_API_KEY`)，构建时注入

**用户 API Key 存储位置**: localStorage (`minimax_api_key`)

**优先级**: 用户配置的 API Key > 默认 API Key

**配置页面**: "我的"页面

**UI 元素**:
- API Key 输入框（密码模式，支持显示/隐藏切换）
- 保存按钮
- 清除按钮
- 获取地址链接
- 默认 Key 状态指示（当使用默认 Key 时显示提示）

- 每行显示5个卡片
- 卡片样式与占卜结果卡片一致
- 响应式断点：
  - ≤1024px: 4列
  - ≤768px: 3列
  - ≤480px: 2列

### 4.6 单屏可见性规范

**核心原则**：用户应能在**不滚动屏幕**的情况下看到页面的所有核心元素和操作按钮。

#### 导航条规范
当页面内容超出视口时，显示**导航条**提示用户滚动：

- 位置：右侧边缘，固定定位
- 宽度：8px
- 样式：金色渐变从上到下
- 圆角：左右圆角 4px

#### 各页面首屏要求

| 页面 | 必须首屏可见 | 可折叠/次屏 |
|------|------------|----------|
| 首页 | 标题、副标题、开始按钮、核心功能卡片(4个) | - |
| 占卜-选牌阵 | 标题、所有牌阵选项(5个)、确认按钮 | - |
| 占卜-问题 | 标题、问题输入框、返回/开始按钮 | - |
| 占卜-抽牌 | 牌堆、进度指示、已抽卡区、按钮 | - |
| 占卜-结果 | 标题、所有卡牌结果、操作按钮 | - |
| 牌库 | 搜索框、筛选器、卡牌网格 | 卡片详情弹窗 |
| 运势 | 标题、所有星座按钮、运势内容 | - |
| 我的 | 登录表单或用户信息 | - |

### 5.3 数据模型

```javascript
{
  id: Number,              // 0-77
  name: String,             // 牌名称
  nameEn: String,           // 英文名
  arcana: String,           // 'major' | 'minor'
  suit: String,             // 'wands'|'cups'|'swords'|'pentacles'|null
  number: Number,           // 1-10 或 null
  imageUrl: String,         // 远程图片URL (备用)

  description: String,      // 牌义描述
  keywords: [String],       // 关键词标签

}
```
